# Добавление кнопок в раздел "Другие обработки"

## Как добавить новую кнопку

Откройте файл `config/config.yaml` и найдите секцию `other_processing_buttons`.

### Пример добавления кнопки:

```yaml
other_processing_buttons:
  - text: "🎄 Елочные игрушки"
    url: "https://t.me/christmas_decoration_bot"

  - text: "🎨 Винтажные портреты"
    url: "https://t.me/vintage_portrait_bot"

  - text: "🖼️ Картины маслом"
    url: "https://t.me/oil_painting_bot"
```

## Параметры кнопки

| Параметр | Описание | Обязательно |
|----------|----------|-------------|
| `text` | Текст на кнопке (можно использовать эмодзи) | ✅ Да |
| `url` | Ссылка на бот или канал | ✅ Да |

## Советы

### ✅ Рекомендации

1. **Используйте эмодзи** для визуального выделения:
   - 🎄 для новогодних/зимних тематик
   - 🎨 для художественных стилей
   - 🖼️ для портретов и картин
   - ✨ для эффектов

2. **Короткие названия** работают лучше (2-4 слова):
   - ✅ "🎄 Елочные игрушки"
   - ❌ "🎄 Создание винтажных елочных игрушек в стиле СССР"

3. **Проверяйте ссылки** перед добавлением

### ❌ Частые ошибки

1. **Неправильный отступ** в YAML:
   ```yaml
   # ❌ Неправильно
   other_processing_buttons:
   - text: "Текст"
     url: "https://..."

   # ✅ Правильно
   other_processing_buttons:
     - text: "Текст"
       url: "https://..."
   ```

2. **Забытые кавычки** для текста с двоеточием:
   ```yaml
   # ❌ Неправильно
   - text: Текст: описание

   # ✅ Правильно
   - text: "Текст: описание"
   ```

## Применение изменений

После изменения конфига **перезапустите бота**:

```bash
# Docker
docker-compose restart bot

# Локально
# Остановите бота (Ctrl+C) и запустите снова
python -m bot.main
```

## Примеры использования

### Добавление бота с фильтрами

```yaml
- text: "📸 Фотофильтры"
  url: "https://t.me/photo_filters_bot"
```

### Добавление канала с примерами


```yaml

```

**Пример:**
```

✨ Здесь собраны другие популярные обработки...

└──────────────────────────┘

│  🎨 Винтажные портреты   │

┌──────────────────────────┐

└──────────────────────────┘
```
